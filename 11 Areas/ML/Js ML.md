### ✅ JavaScript 딥러닝/신경망 라이브러리 비교 정리

|라이브러리|주요 특징 및 용도|
|---|---|
|**TensorFlow.js**|가장 강력하고 널리 사용되는 JS 딥러닝 라이브러리. 학습과 추론 모두 가능. WebGL을 통한 GPU 가속. 브라우저 및 Node.js 지원. 다양한 사전 학습 모델 제공.|
|**ConvNetJS**|브라우저 기반 CNN 지원. 실시간 학습 및 시각화 가능. 강화학습 모듈 포함. 설치 없이 사용 가능. 학습/추론 모두 가능하지만 GPU 가속 없음.|
|**Brain.js**|간단한 신경망(FNN, RNN, LSTM 등) 지원. 브라우저와 Node.js 모두 사용 가능. GPU 가속 지원. 게임 AI, 실시간 예측 등에 적합.|
|**Keras.js**|Python Keras로 훈련된 모델을 브라우저에서 추론용으로 실행. WebGL 기반 GPU 가속. 학습은 불가. 추론 전용.|
|**Synaptic.js**|유연한 신경망 아키텍처 설계 가능. 다양한 커스텀 구조를 직접 구성할 수 있음. 브라우저/Node.js 지원. 학습/추론 모두 가능.|
|**ml5.js**|TensorFlow.js 기반. 초보자 친화적 API. 이미지, 소리, 텍스트 모델 제공. 교육용, 프로토타입에 적합. 사전 학습 모델 활용 중심.|
|**Deeplearn.js**|TensorFlow.js의 전신. 현재는 통합되어 별도 사용 권장되지 않음.|
|**neurojs**|강화학습 포함. 경량 프로젝트에 적합. 실험적인 기능 위주.|
|**PyTorch.js**|아직 초기 단계. 생태계와 문서 부족. PyTorch 기반 모델의 웹 적용 실험용으로 가능성 있음.|

### 📌 기타 관련 라이브러리

|라이브러리|주요 용도|
|---|---|
|**Tesseract.js**|OCR(광학 문자 인식) 기능에 특화. 이미지 내 문자 추출 가능.|
|**OpenCV.js**|컴퓨터 비전 및 영상 처리 기능 제공. 일부 머신러닝 알고리즘 포함.|
|**Mind.js**|간단한 신경망 학습 및 추론. 데이터 배열 연산 기능 지원.|
|**Tracking.js**|객체 추적, 얼굴 인식 등 비전 중심 기능 제공.|

### 📝 요약 가이드

- **학습 + 추론 모두 가능**: TensorFlow.js, ConvNetJS, Brain.js, Synaptic.js
    
- **추론 전용**: Keras.js
    
- **초보자/교육용**: ml5.js
    
- **PyTorch 생태계 실험**: PyTorch.js
    
- **경량/실험용**: neurojs
    
- **비전/텍스트 특화**: OpenCV.js, Tesseract.js
