# 🏠 Areas (Responsibilities)

## Purpose
지속적인 관리와 유지가 필요한 '책임 영역'을 다룹니다.
프로젝트와 달리 마감기한(Deadline)이 없으며, 평생에 걸쳐 신경 써야 하는 부분들입니다.
(PARA 방법론의 'A')

## Examples
- `건강 (Health)`
- `재정 (Finance)`
- `자기계발 (Self-Dev)`
- `집 관리 (Home)`
- `육아 (Parenting)`

## Difference from Projects
- **프로젝트:** "앱 출시하기" (마감일 있음)
- **영역:** "앱 개발 스킬 유지하기" (계속되는 활동)
